# 多人配音工具 · 使用教程（图文）

> 适用版本：**v1.0.0**  
> 仓库：https://github.com/shukeCyp/VoiceGen  
> 示意短视频见：`docs/assets/tutorial-preview.mp4`（概念演示，非真实录屏）

---

## 1. 这是什么？

**多人配音工具（VoiceGen）** 是一个桌面软件，用来：

1. 把「多角色台词表」用不同**本地参考音色**克隆成语音  
2. 可选 **直译 / 本土化**（大模型）  
3. 一键合成 **整轨 MP3**

技术要点（了解即可）：

| 能力 | 调用 |
|------|------|
| 配音 TTS | MiMo `mimo-v2.5-tts-voiceclone` |
| 直译 / 本土化 | MiMo `mimo-v2.5-pro` → `POST …/chat/completions` |
| 默认接口 | `https://token-plan-cn.xiaomimimo.com/v1` |

## 2. 获取与启动

### 方式 A：Windows 安装包（推荐给最终用户）

1. 打开仓库 Actions：https://github.com/shukeCyp/VoiceGen/actions  
2. 选最新成功的 **Build Windows EXE**  
3. 下载 Artifacts 里的 **`多人配音工具.exe`**  
4. 放到任意文件夹运行  

**exe 同目录**会自动使用 / 生成：

```text
多人配音工具.exe
voices/          ← 放参考音色
output/          ← 生成的 MP3
data/            ← 配置与日志（含 data/log）
```

> Windows 包已内置 **ffmpeg / ffprobe**，一般不必再装。需要 WebView2（Win10/11 通常自带）。

### 方式 B：开发模式（macOS / 本机源码）

```bash
cd VoiceGen
# 可选：export MIMO_API_KEY="tp-你的密钥"
./run.sh
# 或：source .venv/bin/activate && python main.py
```

## 3. 第一次使用：三分钟走通

### 步骤 ① 配置 API

1. 打开左侧 **设置**  
2. 填写 **MiMo API Key**（`tp-…` Token Plan）  
3. Base URL 默认：`https://token-plan-cn.xiaomimimo.com/v1`  
4. 点 **检测 API**，看到「TTS+LLM 就绪」即可  
5. 右下角悬浮 **保存设置**

> 启动时若已配置 Key，会**自动检测**一次。  
> 若出现 `WinError 10061 积极拒绝`，多半是本机代理开了但未启动，请关代理或打开代理软件。

### 步骤 ② 准备音色

1. 左侧进 **音色**  
2. 点 **导入音色**，或把文件丢进 `voices/`  
3. 支持 wav / mp3 等；**文件名（无扩展名）= 音色名**  
4. 每条可点 **试听** 听参考音  

示例：

```text
voices/Alex.mp3
voices/Natasha.mp3
voices/德语.mp3
voices/法语.mp3
```

### 步骤 ③ 写台词 / 导入表格

1. 左侧回 **配音**  
2. 直接填表格，或 **导入表格**  
3. 可用 **导出模板** 拿到中文表头样例  

表头（中文）：

```text
角色,音色,原文,配音文本,语速,风格,启用
```

| 列 | 含义 |
|----|------|
| 角色 | 显示名，如 甲 / 旁白 |
| 音色 | 对应 voices 里的名字，如 Alex |
| 原文 | 源语言台词 |
| 配音文本 | 真正拿去 TTS 的文本（翻译结果写这里） |
| 语速 | 如 1.0、1.05 |
| 风格 | 可选风格提示 |
| 启用 | 是否参与生成 |

### 步骤 ④（可选）翻译

1. 选好 **源语言**、**目标语言**  
2. **直译**：忠实翻译（`mimo-v2.5-pro`）  
3. **本土化**：按目标市场口语改写  
4. 结果会**分批实时**写进「配音文本」列  

源语言 = 目标语言时，不调接口，直接复制原文。

### 步骤 ⑤ 生成整轨 MP3

1. 表头复选框可 **全选/取消全选**  
2. 确认每行有音色 + 文本  
3. 右上角 **生成 MP3**  
4. 支持多线程并发（设置里可调线程数）  
5. 完成后可在底部打开输出路径；`output/` 下也有文件  

### 步骤 ⑥ 试听与单条重做

| 操作 | 位置 |
|------|------|
| 试听本条 | 操作列 🎧 |
| 重新生成本条 | 操作列 刷新 |
| 试听参考音色 | 音色页操作列 |

## 4. 常用界面说明

### 左侧导航

- **配音**：主工作台  
- **音色**：音色库列表  
- **设置**：API、模型、并发、主题、关于  

### 配音页工具栏

- 添加行 / 导入表格 / 导出表格 / 导出 JSON / 导出模板  
- 一键清空 / 输出目录  
- **显示列**：窗口小时可隐藏「语速」等列（会记住选项）  

### 设置页亮点

- 马卡龙换肤  
- TTS 并发线程、翻译并发与批大小  
- 关于：版本号、**打开日志目录**

## 5. 日志与排错

### 日志

- 路径：`data/log/voicegen-YYYY-MM-DD.log`  
- 每天一个，最多保留 **7 天**  
- 含：启动、API、翻译、TTS 每行、ffmpeg、界面关键操作  

### 常见问题

| 现象 | 处理 |
|------|------|
| 检测 API 失败 / 积极拒绝 | 检查网络与系统代理；确认 Base URL |
| 进度条中文乱码 | 使用最新版本（已修 UTF-8） |
| 没有 ffmpeg | 用 GitHub 打包的 exe（已内置）；开发机可跑下载脚本 |
| 窗口太小列挤没 | 点「显示列」关掉不需要的列 |
| 翻译慢 | 走大模型 `mimo-v2.5-pro`；可调翻译并发/批大小 |

## 6. 推荐工作流（四人相声示例）

1. 四个参考音放进 `voices/`（如 Alex / Natasha / 德语 / 法语）  
2. 导入已写好的 `voicegen_template.csv` 或自建表格  
3. 中文配音：源=目标=中文，可直接生成  
4. 要外语：选目标语言 → 直译或本土化 → 检查配音文本 → 生成  
5. 在 `output/` 取 MP3  

---

## 7. 隐私与安全

- API Key 只存在本机 `data/config.json`，**不要提交到 Git**  
- 日志不写入密钥  
- 音色、成片均在本地目录
